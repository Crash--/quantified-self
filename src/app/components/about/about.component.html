Welcome to Quantified Self v{{ appVersion }}

 an OpenSource project under development Hosted on Google App Engine, made with Angular Source: GitHub

Quantified Self is a tool for importing viewing and comparing tcx, fit and json files from activity trackers and smart-watches such as Garmin, Suunto, Polar etc.

Currently the support is limit to the main formats: GPX, TCX, FIT and JSON*

*JSON is for specific services while GPX, TCX, FIT should be compatible with the move common services, such as Strava, Movescount, Garmin, Polar and any other service that supprots the above exports.

Quantified Self and quantified-self.io do not process any information from visitors as the application runs locally and stores data only on the vistors browser and not on any other service or provider.

Quantified Self however, uses services such as Analytics and Sentry to report:
  • Anatomized non IP and non visitor identifiable data, for the purposes of visit counting.
  • Anatomized non IP and non visitor identifiable data, for the purposes of error alerts.

The above cases are application specific and contain NO USER Data but only following:

  • Browser version (application specific)
  • Country
  • Crash error message such as 'could not find laps' (application specific)
